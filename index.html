Projecten

Dit zijn een paar persoonlijke projectjes.

MC Mod Updater

Ik speel vaak met veel client-side mods, dus als een nieuwe versie van Minecraft uit komt, moet ik die allemaal updaten.
Ik vond het te veel moeite om elke mod op te zoeken, de juiste versie te vinden, te downloaden en dan in het goeie mapje te zetten, dus ik wou een app maken waar je links van mods in een bestand kan zetten, en die dan in één keer kan downloaden.
Het mooie was ook dat alle links werden opgeslagen, dus je hoefde ze niet de hele tijd op te zoeken.
Na dat alle links in een bestand staan, heb je de keuze om de mod loader en versie te kiezen, daarna zoekt de app alle mods op met de CurseForge en Modrinth API.

Server Shutdown Timer

Toen ik met een paar vrienden een Minecraft server had, hostte ik mijn eigen server thuis. De computer mocht niet 24/7 aanstaan van mijn moeder, dus ik had een server-side mod gemaakt die de server automatisch uit zet.
De timer startte op het moment dat er geen spelers meer online waren en stopte wanneer er iemand online kwam voordat de timer af ging.

MC WebPanel

Ik host nu mijn Minecraft server in de cloud, maar ik kan niet makkelijk bij de console.
Dus ik heb een mod gemaakt die een website host met een console, zodat je direct met de server kan praten via de website. Natuurlijk is de website goed beveiligd.
